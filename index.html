VIP与举报管理插件

为Minecraft 1.20.1 Paper服务端打造的轻量级管理工具，特别适合管理员较少的服务器

版本 2.21 支持 Paper 1.20.1
开始使用

核心功能

简单实用的功能设计，满足服务器管理基本需求

举报系统

  • 使用 /report <玩家> <原因> 命令举报违规玩家
  • 支持四种举报原因：作弊、辱骂、利用漏洞、消极游戏
  • 不同违规原因对应不同封禁时长（30天至1小时不等）
  • 30秒举报冷却时间，防止滥用
  • 无法举报管理员(OP)玩家，保护管理权限

VIP管理

  • 管理员通过 /vip <add/remove> <玩家> 管理VIP
  • 添加VIP时需指定等级：/vip add <玩家ID> <等级>
  • VIP玩家举报处理优先级更高（5秒 vs 普通玩家5分钟）
  • 聊天和名称前缀显示特殊标识（如§e[VIP X]）
  • 管理员兼VIP玩家显示§c[Admin]§e[VIP X] 特殊标识

自动处理机制

  • 根据举报原因自动执行对应时长的封禁
  • 全服广播违规处理结果，维护服务器秩序
  • 自动检查插件更新（默认每5小时一次）
  • 基于计分板团队系统的玩家前缀自动管理

灵活配置选项

  • 可自定义各类违规行为的封禁时长
  • 设置VIP最大举报次数和冷却时间
  • 自定义消息模板，统一服务器通知风格
  • 调整更新检查频率，平衡性能与时效性

命令说明

简单直观的命令设计，易于使用和记忆

举报命令

玩家用于举报违规行为的命令

/report <玩家> <原因>

参数说明：

  • <玩家> ：被举报玩家的ID（必须在线）
  • <原因> ：举报原因（必须是以下之一：作弊、辱骂、利用漏洞、消极游戏）

VIP管理命令

管理员用于管理VIP玩家的命令（需要管理员权限）

/vip add <玩家ID> <等级>

添加指定玩家为VIP，并设置等级

/vip remove <玩家>

移除指定玩家的VIP身份

安装步骤

简单几步，快速部署插件

  1. 1

    下载插件

    获取最新版本的Mc1.20.1-VIPandReport插件JAR文件

  2. 2

    放置插件

    将下载的JAR文件放入您的Minecraft 1.20.1 Paper服务端的plugins文件夹中

  3. 3

    启动服务器

    启动或重启您的Paper服务器，插件将自动加载并生成默认配置文件

  4. 4

    配置插件（可选）

    在plugins/VipReport/config.yml中根据您的需求调整配置参数，然后使用/reload命令重载配置

配置说明

通过配置文件自定义插件行为

配置文件位于 plugins/VipReport/config.yml，以下是主要配置项说明：

封禁时长设置

ban-duration: cheat: 2592000000 # 作弊 - 30天（毫秒） abuse: 86400000 # 辱骂 - 1天（毫秒） exploit: 604800000 # 利用漏洞 - 7天（毫秒） afk: 3600000 # 消极游戏 - 1小时（毫秒）

举报设置

report: cooldown: 30 # 举报冷却时间（秒） vip-priority-time: 5 # VIP举报处理时间（秒） normal-priority-time: 300 # 普通玩家举报处理时间（秒）

更新检查

update-check: enabled: true # 是否启用自动更新检查 interval: 18000 # 检查间隔（秒），默认5小时
